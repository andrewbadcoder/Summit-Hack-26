What the app actually does
User points phone camera at an item → app returns:

1. Identification — what it is (e.g., "Lithium-ion 18650 battery")
2. Recyclability verdict — recycle / e-waste dropoff / hazardous / repair-first / trash, with reasoning
3. Shelf life or condition — visible damage, swelling (batteries), corrosion, wear indicators
4. Action steps — nearest dropoff location, manufacturer take-back program, repair guide link
5. Hazard Concerns — What will happen if the user does not dispose of the device properly, notes any safety concerns
